### Other Features
- Search by **UTC offset**: `utc+8`, `utc-5`
- Search by **timezone name**: `asia/tokyo`, `europe/paris`
- Search by **city name**: `sydney`, `mumbai`, `dublin`
- **Partial matches**: Type `aust` to find Austria and Australia
- **Case insensitive**: `US`, `us`, `Us` all work the same
