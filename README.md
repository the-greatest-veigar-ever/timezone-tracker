clau# Global Time Tracker

A beautiful, futuristic timezone tracker for managing meetings across the globe. Built with a "tomorrow-world" aesthetic featuring glassmorphism, smooth animations, and a modern dark theme.

## Features

- **Real-time Clock Updates**: Live time display for multiple timezones
- **Easy Timezone Management**: Add and remove timezones with a simple interface
- **Meeting Time Finder**: Select a time and highlight it across all timezones to find the best meeting slot
- **Persistent Storage**: Your timezone selections are saved locally
- **Responsive Design**: Works beautifully on desktop and mobile
- **Futuristic UI**: Tomorrow-world aesthetics with gradients, glassmorphism, and smooth animations

## Demo

Open `index.html` in your browser or visit the GitHub Pages deployment.

## Deployment to GitHub Pages

1. Create a new repository on GitHub
2. Push this code to your repository:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO.git
   git push -u origin main
   ```
3. Go to your repository settings
4. Navigate to "Pages" section
5. Under "Source", select "main" branch and "/" (root) folder
6. Click "Save"
7. Your site will be available at `https://YOUR_USERNAME.github.io/YOUR_REPO/`
